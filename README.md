## 常见问题

**DMIT 的三网 CN2 GIA 和普通 CN2 GIA 有什么区别？**

普通 CN2 GIA 可能只有电信回程走 GIA，联通和移动走其他线路。DMIT 的 EB 系列是三网都有优化——电信和联通走 CN2 GIA，移动走 AS4837，整体比只优化单网的方案均衡很多。Pro 系列则是三网全程 CN2 GIA，线路更纯粹。

**流量双向计算是什么意思，会不会很快用完？**

就是你下载 1GB 算1GB，上传 1GB 也算 1GB，两者加起来计入总流量。如果只是日常浏览、代理使用，300GB 或 600GB 的套餐基本够用。如果有大量上传需求，建议选流量更大的方案。

**DMIT 支持退款吗？**

官方提供购买后 72 小时内的退款政策，超过时间窗口后不支持无理由退款。下单前建议先确认好自己的需求。

**洛杉矶 EB 和 Pro 系列，普通用户选哪个？**

大多数人选 EB 就够了。EB 的三网优化已经很到位，价格也低一半以上。Pro 系列适合对线路纯度有极高要求、或者业务对稳定性非常敏感的用户。

**DMIT 的 IP 容易被封吗？**

相比一些超售严重的服务商，DMIT 的 IP 质量整体偏好。但没有任何服务商能保证 IP 永远不被封，这是行业普遍情况。遇到 IP 问题可以联系客服处理。

**香港节点值得选吗？**

香港节点延迟比洛杉矶低很多，对大陆用户来说物理距离更近。但价格也明显更高，且库存经常紧张。如果预算充足、对延迟极度敏感，香港是更好的选择。
